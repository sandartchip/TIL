

- makemigrations : 모델의 변경 사항을 저장하고, 모델의 변경에 따른 SQL 생성 
- migrate : 해당 SQL을 DB에 적용
(Git의 커밋, push와 비슷)

- 마이그레이션은 각 앱마다 따로 된다.

## 특정 시점으로 돌아가기
```
python manage.py migrate [앱 이름] 0001_initial
```
