
- 암호화된 데이터 삽입 

```sql

INSERT INTO TEST_TABLE (name) VALUES (HEX(AES_ENCRYPT('abc', 'KEY VAUE')));

```

- 암호화된 데이터 조회 
```sql
SELECT AES_DECRYPT(UNHEX(COLUMN_NAME), 'KEY VALUE') FROM table_name;
```

위의 암호화 예제를 복호화

```sql
SELECT AES_DECRYPT(UNHEX(name), 'key') FROM test; 
```
-> abc 가 나옴.
