# WSGI

- WSGI는 Web Server Gateway Interafce의 약자.
- 웹 서버와 외부 프로그램 사이에서 정보를 주고받는 방법 

- WSGI는 파이썬 애플리케이션이 웹서버(아파치, NGINX) 와 통신하기 위한 표준을 정의한 것. 
- gunicorn, uwsgi 등이 있음.
