
#### 입력 텐서를 값으로 채움
```python
torch.nn.init.constant_(tensor, val)
```  

```python
 w = torch.empty(3, 5)
 nn.init.constant_(w, 0.3)
```

#### input tensor를 uniform distribution(a, b 범위에서 모든 값이 균일한 분포)로 초기화
```python
nn.init.uniform_(input, a=0.0, b=0.1)
```
