

#### input tensor를 uniform distribution(a, b 범위에서 모든 값이 균일한 분포)로 초기화
```python
nn.init.uniform_(input, a=0.0, b=0.1)
```

#### input tensor가 He 초기값을 N(0, std^2)의 정규분포를 갖는다
```python
nn.init.kaiming_normal_(input, mode='fan_in', nonlinearity='relu')
```
