


```python
a = np.array([1.5, 0.2, 4.2, 2.5])
s = a.argsort()

print(s)  
-> [1 0 3 2]

print(a[s])
-> [0.2, 1.5, 2.5, 4.2]
```

```a.argsort()```는 어레이 a를 정렬하는 인덱스의 어레이 [1 0 3 2]를 반환함.

a[s]와 같이 인덱스의 어레이 s를 사용해서 어레이 a를 다시 정렬하면, 오름차순으로 정렬된 어레이 [0.2, 1.5, 2.5, 4.2]가 됨.
