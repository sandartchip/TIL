

```python
a = np.array([[1, 2], [3, 4]])
```

```
np.mean(a)
```
-> 전체가 평균됨 
-> 2.5


```python
np.mean(a, axis=0)
```
-> axis가 0이므로, 맨 가쪽 축 기준으로 평균->뒤에꺼-> 옆으로가 아닌 세로임 -> 1과 3, 2와 4
-> 1+3 /2, 2+4/2
-> array([2., 3.])

```python
np.mean(a, axis=1)
```
-> 두 번째 축-> 맨 가쪽 축이 아닌 축. -> 안쪽 축 -> 옆으로 평균
-> 1+2/2, 3+4/2 
-> [1.5, 3.5]
