
## reset
reset - > 커밋 이력이 삭제됨. 

### reset --hard
- 이전 버전으로 파일들이 돌아감

### reset --soft
- 내 컴퓨터의 파일 자체는 안 바뀜 
- 커밋만 이전으로 돌아감
- staging area 안바뀜

### reset --mixed 
- 내 컴퓨터의 파일 자체는 안 바뀜
- 커밋은 이전으로 돌아감
- staging area  파일은 이전으로 돌아감

### reset --hard 
- **내 컴퓨터 파일이 바뀜**
- 커밋은 이전으로 돌아감
- staging area 파일 역시 이전으로 돌아감 

## revert
revert -> 현재의 이력을 남기고 과거로 돌아감.
