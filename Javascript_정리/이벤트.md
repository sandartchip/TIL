
# 파일 변경 이벤트

change 이벤트는 
 - input, select, textarea에서, 사용자에 의해 요소의 값이 변경이 되면 fire된다. 

 - change 이벤트는, 각 요소에 따라 다르게 반응한다. 
 예) 사용자가, 변경 사항을 명시적으로 줄 때 
  - ex1.  <input type="file">의 경우, file picker에서 파일을 선택할 때. 
  - ex2.  
 ```html <input type="radio">```
  에서 checked가 됬을 때.
