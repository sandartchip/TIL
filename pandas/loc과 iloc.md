

### loc
#### 레이블 이름이 1인 행에 접근
df.loc[1]

이 때 loc[1]에서 1은 레이블 이름이므로 인덱스 번호와는 구분됨. 

#### 레이블 이름이 city인 열에 접금
df.loc[:, "city"]


### iloc

#### 첫번째 행 
```
df.iloc[0]
```

#### 마지막 행 
```
df.iloc[-1]
```

#### 첫번재 열
```
df.iloc[:, 0]
```
- :은 전체 행을 말하고, 0은 column index. 

#### 두번째 열
```
df.iloc[:, 1]
```
