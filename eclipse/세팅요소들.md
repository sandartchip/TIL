### context root(path)

WAS(Web application service)에서 
웹 어플리케이션을 구분하기 위한 path

예시)

```
localhost:8080/context_root/main.do
```

### 변경방법


1. 프로젝트 우클릭 ->  [Properties] - [Web Project Settings]에 Context root에서 설정 후 적용
