

# width 속성

## width : auto 
- width 지정 안하면, default가 auto 
- auto : 요소의 너비를 브라우저가 계산. 내용

### block 요소의 width auto 
- ** width auto ** : **부모 요소 너비까지 최대값.** 내용물에 맞게 자동으로 너비 조정(X)
- 부모 요소의 너비를 초과할 수는 없기 때문에, 부모 요소의 너비가 되는 것이다.
